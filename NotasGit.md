en git son

* git init

   * Se utiliza para ver el

* git show 
   * Te muestra los cambios que
   hiciste en el ultimo commit
* git show commit_id
  * Te muestra los cambios que
  hiciste en el commit dado

Para poner un parrafo en negritas
hace falta rodear a la palabra con dos **. Ejemplo:
```md

```
